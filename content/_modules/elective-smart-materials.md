This course covers shape memory alloys, piezoelectric ceramics, magnetostrictive materials, and electro-active polymers. Students learn the thermodynamic couplings behind these behaviors and design smart actuators/sensors.

### Key Topics Covered:
- Martensitic phase transformations in shape memory crystals
- Piezoelectric coefficients and sensor circuitry design
- Magneto-rheological fluids and vibration dampers
- Smart skins, self-healing coatings, and flexible sensors
